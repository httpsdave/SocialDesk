## Getting Started

### Prerequisites

- Node.js 18+ and npm

### Installation

1. Clone the repository:
```bash
git clone https://github.com/YOUR_USERNAME/SocialDesk.git
cd SocialDesk
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:5173`

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build

## Deployment

This project is optimized for deployment on Vercel. Simply connect your GitHub repository to Vercel for automatic deployments.

## Routes

- `/` - Login page
- `/app` - Dashboard (main view with stats and charts)
- `/app/schedule` - Post scheduling interface
- `/app/analytics` - Analytics and insights
- `/app/settings` - App settings
- `/app/profile` - User profile
- `/app/add-account` - Add new social media accounts
